CSS Transitions & Animations

Property Explanation Example
transition Smoothly animates property changes like color, size, position. E.g. transition: all 0.5s ease;
Hover me!
.animate-example {
  transition: all 0.5s ease;
  animation: pulse 2s infinite;
}

.animate-example:hover {
  background-color: #28a745;
  transform: scale(1.2) rotate(5deg);
}
transform Changes shape, size, or position of an element. E.g. transform: scale(1.2) rotate(5deg);
animation Runs keyframe animations to create complex effects. E.g. pulsing shadows or moving elements.
