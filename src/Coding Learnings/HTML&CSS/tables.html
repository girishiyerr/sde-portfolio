Table Attributes

Attribute Why it matters Example
colspan Merges cells across multiple columns. Useful for headings or grouped data.
Merged Header
Cell 1 Cell 2
<table>
   <tr>
    <td colspan="2">Merged Header</td>
  </tr>
  <tr>
    <td>Cell 1</td>
    <td>Cell 2</td>
  </tr>
</table>
rowspan Merges cells across multiple rows. Useful for vertical grouping of data.
Merged Left Row 1
Row 2
<table>
  <tr>
    <td rowspan="2">Merged Left</td>
    <td>Row 1</td>
  </tr>
  <tr>
    <td>Row 2</td>
  </tr>
</table>
