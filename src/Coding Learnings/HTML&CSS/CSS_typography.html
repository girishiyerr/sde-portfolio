CSS Typography

Property Explanation Example
font-family Defines the font style used for text. E.g. Arial, sans-serif or 'Georgia', serif.
This is a typography example!
.typography-example {
  font-family: 'Georgia', serif;
  font-size: 24px;
  font-weight: 700;
  line-height: 1.5;
}
font-size Controls the size of text (e.g. 16px, 1.5em, 2rem).
font-weight Sets the thickness of text. Values like normal, bold, or numbers (e.g. 700).
line-height Controls the space between lines of text. Higher values mean more spacing.
