CSS Color & Visuals

Property Explanation Example
color Sets the color of text. Accepts names, hex, RGB, HSL etc.
This is a visuals example!
.visuals-example {
  color: white;
  background-color: #007bff;
  background-image: linear-gradient(135deg, rgba(255,255,255,0.2) 25%, transparent 25%, transparent 50%, rgba(255,255,255,0.2) 50%, rgba(255,255,255,0.2) 75%, transparent 75%, transparent);
  background-size: 40px 40px;
  box-shadow: 0 8px 20px rgba(0, 0, 0, 0.3);
  opacity: 0.9;
}
background-color Sets a solid background color for an element.
background-image Places an image or gradient behind an element’s content.
box-shadow Adds shadow around an element for depth. Format: offset-x offset-y blur color.
opacity Makes elements semi-transparent (from 0 = invisible to 1 = solid).
