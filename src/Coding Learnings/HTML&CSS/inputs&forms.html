Inputs & Forms Attributes

Attribute Why it matters Example
type Defines the behavior of the input, e.g. text, password, email, number, etc. <input type="email" placeholder="Enter your email">
name Used as the key when submitting form data. Identifies the input in the server. <input type="text" name="username" placeholder="Username">
placeholder Displays hint text inside the input box before the user types. <input type="text" placeholder="Search here...">
value Pre-fills the input with a preset value. <input type="text" value="Girish">
required Makes the field mandatory before form submission. <input type="text" placeholder="Required field" required>
checked Sets a radio button or checkbox to be selected by default. <input type="checkbox" checked> Subscribe
disabled Makes the input uneditable and visually grayed out. <input type="text" value="Can't edit me" disabled>
