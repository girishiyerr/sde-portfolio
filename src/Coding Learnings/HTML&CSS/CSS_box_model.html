CSS Box Model

Property Explanation Example
Box Model The box model defines how every element occupies space:
  • Content – the text or image inside the box
  • Padding – space between content and border
  • Border – the line around the element
  • Margin – space outside the border, separates elements
  • box-sizing – decides if width includes padding/border
I am a box!
.box-example {
  width: 200px;
  padding: 20px;
  border: 5px solid #333;
  margin: 30px auto;
  box-sizing: border-box;
}
