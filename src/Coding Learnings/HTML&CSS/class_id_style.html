Universal HTML Attributes

Attribute Why it matters Example
class Used to apply CSS styles or target elements in JavaScript. Multiple elements can share the same class. <div class="box">...</div>
id Used for unique identification. Helps in JS manipulation or anchor navigation. Should be unique on the page. <div id="main-header">...</div>
style Applies inline CSS directly to the element. Not recommended for large styling, but good for quick tweaks. <p style="color: red;">Hello</p>
