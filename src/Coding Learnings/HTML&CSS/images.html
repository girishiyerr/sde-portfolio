Image Attributes

Attribute Why it matters Example
src Specifies the image file path or URL to load the image. <img src="https://via.placeholder.com/150" alt="Placeholder Image">
alt Provides alternative text for screen readers, improves accessibility, and helps SEO. <img src="https://via.placeholder.com/150" alt="A simple placeholder image">
width / height Controls the size of the displayed image. Can use pixels or percentages. <img src="https://via.placeholder.com/300" width="100" height="100" alt="Resized image">
