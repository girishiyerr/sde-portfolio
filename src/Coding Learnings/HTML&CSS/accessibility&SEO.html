Accessibility & SEO Attributes

Attribute Why it matters Example
aria-* Used for screen readers to describe elements. Helps users with disabilities navigate the page. <button aria-label="Close this popup">❌</button>
title Shows a small tooltip on hover. Can provide extra context or accessibility hints. Hover over me! <span title="This is a tooltip example">Hover over me!</span>
lang Sets the language of the document for screen readers and SEO. Helps search engines and browsers interpret the content correctly. <html lang="en">
alt Alternative text for images. Helps visually impaired users and improves SEO if images don’t load. <img src="https://via.placeholder.com/150" alt="Placeholder example image">
