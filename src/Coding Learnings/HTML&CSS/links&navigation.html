Links & Navigation Attributes

Attribute Why it matters Example
href Specifies the URL where the link goes. Can be absolute (full URL) or relative (local file). Visit Example <a href="https://www.example.com">Visit Example</a>
target Determines how the link opens. E.g. _blank opens in a new tab. Open Example in New Tab <a href="https://www.example.com" target="_blank">Open Example</a>
rel Defines the relationship between this page and the linked page. E.g. noopener noreferrer improves security with target="_blank". Secure Example Link <a href="https://www.example.com" target="_blank" rel="noopener noreferrer">Secure Example Link</a>
