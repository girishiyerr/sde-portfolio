CSS Responsive Design

Property Explanation Example
@media queries Rules that apply styles only under certain screen conditions (width, height, orientation, etc.). E.g. @media (max-width: 600px) { ... }
Resize your browser!
.responsive-example {
  background-color: #007bff;
  font-size: 24px;
}

@media (max-width: 600px) {
  .responsive-example {
    background-color: #28a745;
    font-size: 18px;
  }
}
max-width / min-width Limits how wide or narrow an element or layout can grow. E.g. max-width: 600px; keeps boxes readable on big screens.
