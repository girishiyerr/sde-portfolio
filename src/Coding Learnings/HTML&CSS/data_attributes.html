Data Attributes

Attribute Why it matters Example
data-* Lets you store custom data on HTML elements for JavaScript to read. Helps keep your HTML clean and separate from logic.
<button data-user-id="12345">Click me!</button>

// JavaScript:
const btn = document.querySelector('.data-btn');
btn.addEventListener('click', () => {
  const userId = btn.dataset.userId;
  console.log(userId);
});
